# 使用方式

## 双开门：启用 / 禁用 MOD

- 主界面分为左右两扇“门”：左侧是「未启用」的 MOD，右侧是「已启用」的 MOD。
- 点击卡片上的 `启用 ▶` / `◀ 禁用` 按钮，或直接双击卡片，即可把 MOD 在两扇门之间移动，并立即静默保存。
- 每扇门标题会显示该侧 MOD 数量，右上角显示 `已启用数量/全部数量`。
- 同名冲突（mods 与 mods_disabled 同时存在）会在卡片上标出 `⚠ 同名冲突`。
- 点击 MOD 名称：打开详情窗口，可查看 JSON 元数据、修改文件夹名、编辑备注。

## 右侧功能列（MOD 操作）

- `刷新`：重新扫描 `mods` / `mods_disabled`；在窗口重新获得焦点时也会自动检测磁盘变化。
- `添加 mod`：从 `.zip` 压缩包或文件夹添加一个新 MOD，自动解析其目录结构与名称，默认加入「未启用」。
- `更新 mod`：先在列表中选中一个 MOD，再选择压缩包/文件夹，用新内容替换它（保留原文件夹名与启用状态）。
- `卸载 mod`：选中一个 MOD 后从磁盘删除它，并清理其备注与配置组合引用。
- `打开 mods` / `打开 mods_disabled`：打开对应的 MOD 文件夹。

## 配置组合

- 配置组合下拉框会自动匹配当前启用的 MOD；不符合任何组合时显示为空白。
- `保存当前为组合`：把当前启用的 MOD 保存成一个配置组合。
- `应用`：精确应用当前选中的配置组合，组合外的 MOD 会被禁用。
- `删除组合`：删除当前选中的配置组合。

## 底部

- `启动游戏`：启动游戏；管理器会在后台等待游戏退出后再结束自身进程。
- `关闭`：退出管理器。

## 首次启动

- 如果是从 Steam 启动项进入，管理器会优先读取 Steam 传入的游戏启动参数。
- 如果没有 Steam 参数，但本地已有上次成功启动的游戏路径记录，管理器会直接使用该记录。
- 如果两者都没有，或者记录路径已失效，管理器会弹出启动配置窗口：
  - 可以手动选择 `Slay the Spire 2` 游戏目录。
  - 也可以点击“自动扫描”，按 Steam 注册表、默认 Steam 路径和 `libraryfolders.vdf` 自动查找。
  - 确认后会把游戏路径保存到本地配置，之后可直接复用。
